A few notes about this dumb web toy...

Speech Synthesis

The speech synthesis is accomplished 100% in the browser using meSpeak.js, client-side JavaScript text-to-speech library based on the speak.js project. Speak.js is an Emscripten port of the C++ eSpeak library.

Design

The on-screen keyboard is pure CSS, designed by the talented Kentuckian Dustin Cartwright. The other elements are Bootstrap. Obviously.

Mobile Support

Absolutely not.

Other technologies used to make this

Angular.js, Angular UI (screen name modal), Grunt, Bower, Firebase
